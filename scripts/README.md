# Generating rebar grid .yaml file

First, to generate the rebar grid .yaml file, run

``` 
python make_environment_yaml.py
```

# Generating rebar grid .world file and .json file

Then, to convert the .yaml file into a .world file and a .json file, run

```
python convert_yaml_to_world.py
```
